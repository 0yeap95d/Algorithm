>- 최대값은 규칙성이 쉽게 나타나서 금방 해결할 수 있었음
>- 최소값을 구하면서 2자리 수 이상인 경우 6을 무조건 0으로 바꿔주었는데 반례가 존재
>  - ex) n이 13일때 80이 아닌 68이 되어야함
>- 0 다음으로 작은 숫자가 6보다 크면 가장 앞에 있는 0을 6으로 변경
