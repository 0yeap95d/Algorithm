>- ⏱ 풀이시간 00분
>- 처음에 각각 탈출하는 경우와 두 죄수가 만나서 탈출하는 경우 두 가지를 비교하여 해결하려 했으나 적절한 솔루션이 아니었음
>- 다른 사람의 솔루션을 참고하여 외부에 사람이 있다고 가정한 후 해결할 수 있었음
>- 변수 초기화 문제 때문에 많은 시간이 소요되었음, 테스트 케이스가 여러개 들어오는 경우 초기화에 좀 더 신경써야할듯
