### 알고리즘 분류

### 문제 및 해결

>- ⏱ 풀이시간 28분
>- DP를 이용하여 해결하려 했으나 높은 숫자에서 낮은 숫자로 내려가는 뱀 조건 때문에 BFS를 이용하였음
>- 주사위를 한번 굴릴때 마다 6개의 노드가 queue에 들어간다고 생각하여 메모리 초과가 발생하지 않을까 걱정했지만 푸는 과정에서 방문체크로 인해 생각보다 많지 않음을 깨달음
>- BFS를 이용하여 구현하는 것은 어렵지 않았음
