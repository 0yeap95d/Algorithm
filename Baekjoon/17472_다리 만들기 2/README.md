> https://www.acmicpc.net/problem/17472



### 알고리즘 분류

>- 구현
>- 그래프 이론
>- 그래프 탐색
>- 브루트포스 알고리즘
>- 너비 우선 탐색
>- 깊이 우선 탐색
>- 최소 스패닝 트리



### 문제 및 해결

>- ⏱ 풀이시간 60분 초과
>- 각 섬에 번호를 붙이고 BFS를 이용하여 섬끼리의 거리를 구함
>- 구한 거리를 이용하여 MST를 만들어주면서 다리의 최소길이를 구할 수 있음
>- 조건이 많아서 시간이 오래 걸리지만 문제만 잘 파악하면 어렵지않은 문제
